# Analytics Dashboard

Dashboard de analytics para e-commerce simulado, construido con React y Recharts. Proyecto de portfolio para demostrar visualización de datos, diseño responsive y herramientas frontend modernas.

[Demo en vivo](https://dashboard-analytics-lr.web.app)

## Funcionalidades

- Tarjetas KPI con comparación por período
- Gráficos interactivos (Área, Barras, Donut)
- Modo oscuro
- Selector de período (3M / 6M / 12M)
- Tabla de transacciones ordenable
- Totalmente responsive (layout de cards en mobile)

## Stack Tecnológico

- React 18
- Vite
- Tailwind CSS v4
- Recharts
- lucide-react
- date-fns
- clsx

## Inicio Rápido

```bash
npm install
npm run dev
```
